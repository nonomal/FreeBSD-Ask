# 第 13.3 节 使用 DTrace

首先安装 ksh93：

```
# pkg install ksh93
```
或者

```
# cd /usr/ports/shells/ksh93/ && make install clean
```

接下来安装 dtrace-toolkit：

```
# pkg install sysutils/dtrace-toolkit
```

或者：

```
# cd /usr/ports/sysutils/dtrace-toolkit/ && make install clean
```
